# SplitChain 🌐💸
**A Decentralized Web3 Bill Splitting Application built on the Stellar Blockchain!** 

SplitChain allows you to split bills, share expenses and settle payments trustlessly through Soroban Smart Contracts.

## 🚀 Key Features

* **Soroban Smart Contracts:** Secure and decentralized escrows. 
* **Seamless Payments:** Integrated natively with Albedo & Freighter wallets.
* **Live Dashboard:** Keep track of who owes you, and who you owe. 
* **Transaction History:** All your completed transactions are fully observable on the Stellar Explorer!

## ⚙️ How to Run Locally

You can easily run this interface on your own computer:

1. **Prerequisites**: Make sure you have [Node.js](https://nodejs.org/) installed on your system.
2. **Download the Code**: Clone or download this repository.
3. **Start the App**: Open your terminal in the downloaded folder and run:
   ```bash
   node server.js
   ```
4. **Access It**: Open your web browser and go to `http://127.0.0.1:8080`.
